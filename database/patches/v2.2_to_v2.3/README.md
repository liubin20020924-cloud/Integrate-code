# v2.2_to_v2.3 升级包说明

## 概述

此升级包包含从 v2.2 版本升级到 v2.3 版本所需的所有数据库补丁。

## 版本信息

- **起始版本**: v2.2
- **目标版本**: v2.3
- **发布日期**: 待定
- **兼容数据库**: MariaDB/MySQL 5.7+

## 补丁列表

当前版本暂无补丁,此目录为后续升级预留。

当需要发布新版本时,请按照以下规范添加补丁:

1. 按序号创建补丁文件: `001_description.sql`, `002_description.sql` ...
2. 补丁文件必须包含详细的注释说明
3. 补丁必须具备幂等性(可重复执行)
4. 在本README中更新补丁列表和说明
